BEM "Block Element Modifer"

The Block Element Modifier methodology is a populer naming convention for classes in both HTML and CSS. It was created to help developers better understand the realtionship between HTML and CSS while working though projects. It was developed by the team at Yandex.

Within this methodology a block would be considered a top level abstraction for a new component, for example if we were designing a button we would name our class: .btn. This block should be thought of as the parent. Child elements can be placed inside and these are represented by two underscores like so: .btn__price. Lastly we will deal with the modifier. The block .btn can be modified in many ways in order to alter the style of a particular compnenet without infilicting changes on an unrelated class. This is done by adding gtwo hyphen to the name of the block level element like so: .btn--orange.
