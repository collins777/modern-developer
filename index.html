Hello

Throughout this project we will explore different ways to level up your developement process. Some of the technologies we will be going over are modern css preproccesing, implementing open source code using package managers , automating workflow with javascript build tools, and industry standard naming conventions. Adopting some of these principals will improve the qaulity of your code allowing for easily maintable and clean code.

In this project I will specifically address the tools I use on a regular basis. As a css preproccessor I generally choose to stick with SASS which stands for Systematically Awesome Style Sheets. In order to implement preproccessors one must use either a build tool, package manger, or both. For these I tend to use NPM "Node Package Manager" and Gulp.js. In order to write cleaner code I use a naming convention called BEM "Block Element Modifier". Thank you for reading and I hope that you find this information useful.
